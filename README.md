# 🔢 FPGA Binary Counter Simulator

An interactive FPGA-style binary counter simulator built using **Python and Streamlit**.

This project recreates the behaviour of a digital binary counter similar to one implemented in **VHDL on an FPGA**, including configurable bit width, counting direction, variable step size, overflow detection, and real-time LED visualization.

The simulator provides a hardware-inspired interface where users can observe binary states, decimal values, hexadecimal outputs, and counter behaviour in real time.

# 📌 Features

## Binary Counter Engine

- Supports configurable:
  - 4-bit counter
  - 8-bit counter

- Counting modes:
  - Up counter
  - Down counter

- Adjustable step size:
  - +1
  - +2
  - +4
  - +8

- Automatic modulo behaviour similar to FPGA counters
